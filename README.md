<h1>ReactJs Boilerplate</h1>

<h2>Thanks for using my Boilerplate/Template 🙏</h2>

### install dependencies
```
cd ReactJs-BoilerPlate
npm install
```

### run (development)
```
npm start
```

### build (production)
```
npm run build
```

<h4>Boilerplate Features</h4>

1. Pre configured <b>node-sass</b>.
2. React Router ready to use.
